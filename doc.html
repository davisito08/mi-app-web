🚀 Guía de Inicio Rápido

GravityLab es una herramienta potente para crear experimentos visuales. Sigue estos pasos para empezar:

  • Crear una Escena: Usa el formulario principal para definir la gravedad y el fondo.
  • Configurar Gravedad: Ajusta los ejes X e Y para cambiar la dirección de la caída.
  • Añadir Fondos: Puedes usar colores sólidos o URLs de imágenes impactantes.
  • Interactuar: Una vez lanzada la escena, puedes arrastrar los elementos con el ratón.
